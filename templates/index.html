Ship Classification

About The Project:


The purpose of this project is to help a user classify Ships into three categories : Air-Carrier,Cargo-Ship,Cruise-Ship. In this project we have used CNN (Deep Learning Algortihm) to train the model. In dataset,we have used 1500 images to train the model while 352 images to test the model.The model is prepared with the help Of Libraries Keras (Backend: TensorFlow). This whole application is Flask server based web application. Sourcecode is available at GitHub

The author of this project is : Anas Aneeque & Soumya M.
